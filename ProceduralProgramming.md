# ProceduralProgramming
## Swift
#### Does the language support procedural programming?
Swift is protocol, object, functional, imperative and block structed oriented not procedural.
~~~
protocol Bird {
  var name: String { get }
  var canFly: Bool { get }
}

protocol Flyable {
  var airspeedVelocity: Double { get }
}
~~~
